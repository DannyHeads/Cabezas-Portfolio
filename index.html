Project Name

NASA Space Explorer

One-Line Hook

Explore NASA's Astronomy Picture of the Day images through a responsive, searchable date-based gallery.

The Problem

NASA provides years of astronomy images and educational information, but users need a simple way to explore content from specific dates. This project gives students, space enthusiasts, and curious users an accessible way to search NASA's Astronomy Picture of the Day collection.

What I Built

I built a responsive web application that connects to NASA's Astronomy Picture of the Day API. Users can select a date range, retrieve NASA images and information, and explore the results through a dynamically generated gallery.

Tech Stack

HTML5 CSS3 JavaScript NASA APOD API Fetch API JSON

Key Features

  1. Retrieves NASA images, titles, dates, and descriptions from a selected date range.
  2. Prevents future-date searches and automatically corrects invalid date selections.
  3. Displays results in a responsive gallery with loading and error feedback.

What I Learned

Technical skill: I improved my ability to retrieve API data asynchronously, process JSON responses, and dynamically update a webpage.

Non-technical skill: I learned to anticipate common user mistakes and design clearer feedback that makes an application easier to understand.
